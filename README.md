# Quiniela Laredo 2026

App base para quiniela familiar/amigos del Mundial 2026.

## Pasos rápidos
1. Sube estos archivos a GitHub.
2. En Vercel, agrega variables de entorno:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
3. En Supabase, ejecuta el archivo `supabase-schema.sql`.
4. Deploy en Vercel.
